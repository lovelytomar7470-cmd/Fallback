# Fallback
Game me ek chhoti si yellow bird hoti hai jise player ko hawa me udte hue green pipes ke beech se nikalna hota hai. Screen par tap karne se bird upar jump karti hai. Tap na karne par bird neeche girti hai (gravity ke wajah se). Har pipe ke beech se safely nikalne par 1 point milta hai. Agar bird pipe ya ground se takra jaye, to game turant
